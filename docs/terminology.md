# 擇吉名詞解釋

基於 forecasting.hk 網站收集的術語定義。

## 宜忌事項

### 婚嫁相關
| 術語 | 說明 |
|------|------|
| **嫁娶** | 男娶女嫁，舉行結婚大典的吉日 |
| **納采** | 締結婚姻的儀式，受授聘金 |
| **訂盟** | 訂婚儀式的一種，俗稱小聘(訂) |

### 祭祀相關
| 術語 | 說明 |
|------|------|
| **祭祀** | 指祠堂之祭祀，即拜祭祖先或廟寺的祭拜、神明等事 |
| **祈福** | 祈求神明降福或設醮還願之事 |
| **求嗣** | 指向神明祈求後嗣(子孫)之意 |

### 喪葬相關
| 術語 | 說明 |
|------|------|
| **安葬** | 舉行埋葬等儀式 |
| **入殮** | 將屍體放入棺材之意 |
| **移柩** | 行葬儀時，將棺木移出屋外之事 |
| **破土** | 僅指埋葬用的破土，與一般建築房屋的"動土"不同。即"破土"屬陰宅，"動土"屬陽宅也。現今社會上多已濫用屬陰宅擇日時屬陰宅須辨別之。 |

### 建築相關
| 術語 | 說明 |
|------|------|
| **動土** | 建築時，第一次動起鋤頭挖土 |
| **起基** | 建築時，第一次動起鋤頭挖土 |
| **上樑** | 裝上建築物屋頂的樑，同架馬 |
| **安床** | 指安置睡床臥鋪之意 |
| **安門** | 放置正門門框 |
| **修造** | 指陽宅之造與修理 |
| **拆卸** | 折掉建築物 |

### 入宅相關
| 術語 | 說明 |
|------|------|
| **入宅** | 即遷入新宅，所謂"新居落成典禮"也 |
| **移徙** | 指搬家遷移住所之意 |

### 商業相關
| 術語 | 說明 |
|------|------|
| **開市** | "開業"之意，商品行號開張做生意"開幕典禮""開工"同。包括：(1)年頭初開始營業或開工等事(2)新設店鋪商行或新廠開幕等事 |
| **立券** | 訂立各種契約互相買賣之事 |
| **交易** | 訂立各種契約互相買賣之事 |
| **納財** | 購屋產業、進貨、收帳、收租、討債、貸款、五谷入倉等 |

### 出行相關
| 術語 | 說明 |
|------|------|
| **出行** | 指外出旅行、觀光遊覽 |
| **赴任** | 走馬上任 |

### 其他事項
| 術語 | 說明 |
|------|------|
| **開光** | 佛像塑成後，供奉上位之事 |
| **栽種** | 種植物"接果""種田禾"同 |

## 沖煞說明

### 六沖
每日地支與特定地支相沖：
- 子沖午（鼠沖馬）
- 丑沖未（牛沖羊）
- 寅沖申（虎沖猴）
- 卯沖酉（兔沖雞）
- 辰沖戌（龍沖狗）
- 巳沖亥（蛇沖豬）

### 煞方
四正方位的凶煞：
- **煞東** - 東方不利
- **煞西** - 西方不利
- **煞南** - 南方不利
- **煞北** - 北方不利

## 時辰吉凶評級

| 評級 | 符號 | 說明 |
|------|------|------|
| **吉** | ☆ 吉 | 此時辰適合辦事（橙色/紅色顯示）|
| **凶** | ● 凶 | 此時辰不宜辦事（黑色顯示）|
| **平** | ○ 平 | 此時辰普通（灰色顯示）|
